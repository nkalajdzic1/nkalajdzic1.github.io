What I do

I am a Software Engineer. First and foremost, my primary task as a developer is to develop software solutions which are fast, maintanable and easy to use.

Front end Engineering

Creating intiutive and good looking UI/GUI is a must for me because that is the first user experience of a software solution and is a decisive factor if the user will continue to use the software.

Back end Engineering

If you have a good looking app, but is not optimized and functional, there is no good use of the app! Back end Engineering is my biggest passion because with Back end I always have the opportunity to challenge myself to produce a high value solution that is fast, clean and reliable.

Algorithms

To create fast and reliable software solutions, one needs to know algorithms: from the basic bubble sort to the graph theory and parallel optimization of algorithms. I always find algorithms challenging and interesting. They are a necessity for a Software Engineer.

Software design and testing

Before I get my hands on the implementation, I create a software design to serve and guide me to to the creation of the software solution. This is for me a must because without planning and desiging inforward, a good software cannot be created. After implementing certain functionalities, I always take my time to do some testing for the app. From basic testing to stress testing and beyond so I can make sure everything works how it should.
